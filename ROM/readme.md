# Прошивки для проекта
## Версия с двумя ПЗУ по 8кб (2764, К573РФ4А)
Прошиваем 48k-0 и 48k-1 соответственно.  
Не забываем установить перемычку JP4.

## Версия с одной ПЗУ на 16кб (27128)
Прошиваем 48K.rom и размещаем в панельке "1"

## Версия с одной ПЗУ на 32кб (27256)
Прошиваем 1-SOS-128-48k.BIN и также размещаем в панельке "1"  
Это задел на будущее расширение 128к  
  
* 1-SOS-128-48k(menu) - по сбросу выходит в меню 128к
* 1-SOS-128-48k(DOS) - по сбросу сразу выходит в TR-DOS

При использовании только одной ПЗУ, во вторую панельку можно  
расположить ПЗУ с прошивкой TR-DOS для будущего использования БДИ
